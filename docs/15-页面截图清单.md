# 15. 页面截图清单

> 文档类型：项目素材文档  
> 目标：提前规划截图素材，避免整理文档或汇报材料时临时补图

## 1. 前台页面截图

| 页面 | 建议截图内容 | 用途 |
|---|---|---|
| 首页 | 顶部导航、主视觉、业务闭环区块、推荐商品 | 介绍项目定位与首页设计 |
| 搜索页 | 搜索框、分类筛选、商品列表 | 展示检索能力 |
| 商品详情页 | 商品主图、价格、库存、推荐理由、溯源批次 | 展示商品展示深度 |
| 购物车页 | 商品列表、数量修改、金额汇总 | 展示订单前置流程 |
| 支付页 | 订单信息、金额汇总、支付成功提示 | 展示支付流程 |
| 我的订单页 | 订单进度条、状态时间字段 | 展示状态流转 |
| 个人中心 | 用户信息、地址管理、后台入口 | 展示用户中心能力 |

## 2. 后台页面截图

| 页面 | 建议截图内容 | 用途 |
|---|---|---|
| 仪表盘 | 统计卡片、低库存预警、最近订单、热销商品 | 展示后台概览能力 |
| 商品管理 | 条件筛选、商品表格、上下架或编辑按钮 | 展示运营管理能力 |
| 分类管理 | 分类列表、启停状态、编辑按钮 | 展示基础数据维护 |
| 订单管理 | 筛选区、订单表格、详情抽屉、发货/完成按钮 | 展示履约处理能力 |

## 4. 截图规范建议

1. 所有截图尽量使用同一浏览器和统一分辨率。
2. 首页、后台仪表盘建议使用宽屏截图，适合宽屏展示和文档排版。
3. 弹窗、抽屉、成功提示要尽量在截图中可见，增强“系统已运行”的证明力。
4. 如果需要说明不同角色入口，可截登录页中的常用账号提示卡片。
